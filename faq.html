How Do I create a file for submission

As this project is in its prelimiary stages, any submission file will be a text file.
Unsure how to create a text file?

  1. Copy the entire email thread with command CTRL-C on your keyboard
    • For Windows users: Right Click on your Desktop and hover to the new selection for Text Document
    • For Mac: Open TextEdit and create a plain text file
  2. Paste contents with CTRL-V. Make sure all the thread was included for best accuracy
  3. Save the text file under any name. For Mac users, make sure the file is txt and not rtf

What information can I acesss once I log in ?

The Adminstrator of your group threads:

  • The master transcript of the analyzed email thread
  • Your 1:1 chats with other memembers
  • The original copy of the submission file
  • Ability to delete the threads from the database
As non-admin users:
  • The analyzed group chat that the user is CC'D in
  • Any 1:1: messages with any group memebers in the submission file

I submitted my file and the browser received a error, What happen?

The transimission from the webpage to the analysis program failed. This may be from: submission file's type, the current network is unstable, and etc. Please pay attention to the error message given and retry submission.
